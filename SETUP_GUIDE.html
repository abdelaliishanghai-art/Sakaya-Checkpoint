أيقونة التطبيق
1
إنشاء حساب GitHub مجاني
  • افتح المتصفح وادخل على github.com
  • اضغط "Sign up" وأنشئ حساباً مجانياً باستخدام أي بريد إلكتروني
  • أكّد بريدك الإلكتروني من الرسالة التي تصلك
2
إنشاء مستودع جديد
  • بعد تسجيل الدخول، اضغط على زر "+" في الأعلى ثم "New repository"
  • في خانة Repository name اكتب: sakaya-guard
  • اختر "Public" (مهم — لازم public لكي تعمل GitHub Pages)
  • اضغط "Create repository"
⚠️ مهم: اسم المستودع يجب أن يكون sakaya-guard بالضبط بدون مسافات
3
رفع الملفات الأربعة
  • في صفحة المستودع الجديد، اضغط "uploading an existing file" أو "Add file → Upload files"
  • اسحب وأفلت الملفات الأربعة كلها دفعة واحدة: index.html · manifest.json · sw.js · icon-192.png · icon-512.png
  • في الأسفل، اضغط "Commit changes" (الزر الأخضر)
4
تفعيل GitHub Pages
  • في صفحة المستودع، اضغط على "Settings" (في القائمة العلوية)
  • من القائمة الجانبية اختر "Pages"
  • تحت "Source" اختر: Deploy from a branch
  • تحت "Branch" اختر: main ثم / (root)
  • اضغط "Save"
  • انتظر دقيقة واحدة، ثم أعد تحميل الصفحة
✅ بعد الحفظ سيظهر رابط مثل: https://اسمك.github.io/sakaya-guard
هذا هو رابط التطبيق — احفظه وأرسله للحراس
5
تثبيت التطبيق على هواتف الحراس
  • أرسل الرابط للحارس عبر واتساب
  • يفتح الرابط على هاتفه
  • Android (Chrome): ستظهر رسالة "تثبيت التطبيق" تلقائياً — يضغط "تثبيت"
  • iPhone (Safari): يضغط زر المشاركة ↑ ثم "Add to Home Screen" ثم "Add"
  • التطبيق يظهر على الشاشة الرئيسية باسم SAKAYA Guard مع الأيقونة
6
ربط التطبيق بـ Google Sheets
  • عند أول فتح للتطبيق، يظهر شاشة إعداد تطلب رابط Web App
  • الرابط هو نفس رابط Apps Script الذي استخدمته في نظام التصاريح (elevator_access)
  • يُدخل الحارس الرابط مرة واحدة فقط — يُحفظ تلقائياً على الهاتف
  • التطبيق يحمّل قائمة التصاريح ويحتفظ بنسخة محلية للعمل بدون إنترنت
💡 تلميح: أرسل الرابط للحارس مع رابط التطبيق حتى يُدخله بسهولة
⚠️ ملاحظة iPhone: يجب فتح الرابط في Safari تحديداً (ليس Chrome أو غيره) لتظهر خيار "Add to Home Screen"
